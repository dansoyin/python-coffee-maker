# Python Coffee Machine ☕

A simple virtual coffee machine project to practice Object-Oriented Programming (OOP) in Python. 

## What it does
* Shows a coffee menu (Espresso, Latte, Cappuccino).
* Checks if ingredients (water, milk, coffee) are enough.
* Takes coins, calculates total, and gives change.

## How to run
Just open your terminal and run this command:
```bash
python main.py
